# Security
## Path Sandboxing
Restrict tools to run within a sandboxed path.
## Tool Allowlisting
Only allow specific tools to be executed by Ollama.
## API Authentication
Verify API requests with authentication tokens.
## Rate Limiting
Limit the number of tasks that can be processed per minute.
## Output Size Limits
Restrict output size for each task.
## Audit Log
Store all task execution events in an audit log.
## Confirmation Prompts
Prompt users before executing sensitive actions.
