# DED Feasibility Analysis Directory

**Purpose:** Structured feasibility analysis for each major DED program and tool.
Each document evaluates a program across six dimensions before asking whether it
should be built, not whether it can be politically sold.

---

## Feasibility Dimensions (Applied Consistently Across All Docs)

| Dimension | What It Assesses |
|-----------|-----------------|
| **Legal / Constitutional** | Statutory authority, enabling legislation required, constitutional objections and responses |
| **Administrative** | Staffing, systems, data infrastructure, agency capacity, inter-agency dependencies |
| **Economic** | Cost model, revenue offsets, ROI projection, fiscal baseline, macro effects |
| **Political** | Coalition map, opposition, sequencing, legislative vehicle, amendments risk |
| **Implementation** | Phasing, critical path, dependencies, failure modes, go/no-go milestones |
| **International Precedent** | Comparable programs elsewhere, what worked, what failed, what transfers |

Each feasibility doc produces a **Feasibility Rating** on each dimension (Strong / Moderate / Weak / Unknown)
and an overall **Implementation Readiness Score** (Ready / Conditional / Needs Work / Premature).

## How to Read the Feasibility Ratings

**Strong** — The program has clear legal authority, administrative precedent, positive ROI evidence,
  and a viable political coalition. No fundamental obstacles to implementation.

**Moderate** — The program is feasible but requires non-trivial work: new enabling legislation,
  significant administrative build-out, or a political coalition that exists but needs to be assembled.

**Weak** — The program faces a real obstacle in this dimension that is not easily resolved. Not
  disqualifying, but requires specific mitigation.

**Unknown** — Insufficient evidence or analysis to rate. Flags an area needing further research
  before implementation planning proceeds.

---

## Implementation Readiness Scores

**Ready** — Can move to legislative drafting and administrative planning immediately.
  No fatal obstacles. Recommend prioritizing.

**Conditional** — Feasible given specific preconditions (e.g., DED must exist before the
  program can launch; or a threshold political event must occur first).

**Needs Work** — The program concept is sound but the implementation design needs further
  development before it is ready for legislative drafting.

**Premature** — The program is too far ahead of current political, technological, or institutional
  conditions to be actionable in the near term. Document for future sessions; do not prioritize.
