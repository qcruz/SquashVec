## Notes for Adding Tags

- Use `snake_case` for all tag IDs
- Keep display names short (≤25 chars for the tag chip UI)
- Add new tags to both this file AND `app/seed_data.py`:
  - `TAG_DISPLAY` dict
  - `TAG_CATEGORIES` dict (pick the right group)
  - `TAG_DEFAULT_INTERVALS` if the tag has a natural repeat frequency
- Tag IDs must be unique across all categories
- After adding tags, update existing event `tags` lists in `SEED_EVENTS` where relevant
